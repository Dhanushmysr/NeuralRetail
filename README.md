# NeuralRetail

## AI-Powered Sales Intelligence and Predictive Analytics Platform

### Project Overview

NeuralRetail is an AI-powered retail analytics platform developed to help retail businesses make data-driven decisions. The platform performs demand forecasting, customer churn prediction, feature engineering, data validation, and sales analytics using Machine Learning and Data Engineering techniques.
